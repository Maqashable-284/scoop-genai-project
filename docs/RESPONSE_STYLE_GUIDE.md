# Scoop AI рЃЕрЃљрЃбрЃЉрЃЮрЃбрЃўрЃА рЃърЃљрЃАрЃБрЃ«рЃћрЃЉрЃўрЃА рЃАрЃбрЃўрЃџрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃљрЃарЃЎрЃљрЃърЃўрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў (v2)

## рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

### рЃбрЃЮрЃюрЃў рЃЊрЃљ рЃ«рЃЏрЃљ
- **рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃЊрЃљ рЃЏрЃћрЃфрЃюрЃўрЃћрЃарЃћрЃЉрЃљрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў** Рђћ рЃљрЃарЃљ рЃЏрЃљрЃарЃЎрЃћрЃбрЃўрЃюрЃњрЃБрЃџрЃў рЃћрЃюрЃљ, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃцрЃљрЃЦрЃбрЃЮрЃЉрЃарЃўрЃЋрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ
- **рЃЌрЃЉрЃўрЃџрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃљ рЃќрЃћрЃЊрЃЏрЃћрЃбрЃљрЃЊ рЃЏрЃћрЃњрЃЮрЃЉрЃарЃБрЃџрЃў**
- **рЃюрЃБрЃбрЃарЃўрЃфрЃўрЃЮрЃџрЃЮрЃњрЃўрЃА рЃърЃЮрЃќрЃўрЃфрЃўрЃљ** Рђћ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃЋрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ рЃўрЃюрЃцрЃЮрЃарЃЏрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљрЃерЃў, рЃљрЃа рЃЋрЃДрЃўрЃЊрЃўрЃЌ

### рЃарЃљ рЃљрЃа рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ
- рЃќрЃћрЃЊрЃЏрЃћрЃбрЃў рЃћрЃЏрЃЮрЃ»рЃћрЃЉрЃў (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ ­ЪЦЄ­ЪЦѕ­ЪЦЅ рЃарЃљрЃюрЃЪрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ ­ЪЏњ рЃДрЃўрЃЊрЃЋрЃўрЃА рЃџрЃўрЃюрЃЎрЃўрЃАрЃЌрЃЋрЃўрЃА)
- рЃќрЃћрЃЊрЃЏрЃћрЃбрЃў рЃФрЃљрЃ«рЃўрЃџрЃўрЃА рЃюрЃўрЃерЃюрЃћрЃЉрЃў
- рЃЏрЃљрЃарЃЎрЃћрЃбрЃўрЃюрЃњрЃБрЃџрЃў рЃцрЃарЃљрЃќрЃћрЃЉрЃў ("рЃерЃћрЃАрЃљрЃюрЃўрЃерЃюрЃљрЃЋрЃў!", "рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ!", "рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃџрЃљрЃЊ!")
- рЃЉрЃљрЃюрЃљрЃџрЃБрЃарЃў рЃЏрЃўрЃАрЃљрЃџрЃЏрЃћрЃЉрЃћрЃЉрЃў ("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ!")
- **рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃў markdown-рЃўрЃЌ** Рђћ рЃўрЃАрЃўрЃюрЃў рЃљрЃа рЃарЃћрЃюрЃЊрЃћрЃарЃЊрЃћрЃЉрЃљ рЃАрЃгрЃЮрЃарЃљрЃЊ рЃДрЃЋрЃћрЃџрЃљ рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃќрЃћ

```markdown
**рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃЋрЃљ:** рЃърЃарЃЮрЃбрЃћрЃўрЃюрЃў + рЃЏрЃљрЃдрЃљрЃџрЃЎрЃљрЃџрЃЮрЃарЃўрЃБрЃџрЃў рЃАрЃљрЃЎрЃЋрЃћрЃЉрЃў (рЃерЃЋрЃарЃўрЃљ, рЃЉрЃљрЃюрЃљрЃюрЃў, рЃЌрЃ«рЃўрЃџрЃўрЃА рЃЎрЃљрЃарЃљрЃЦрЃў) рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃБрЃцрЃарЃЮ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃњрЃќрЃљ рЃЎрЃљрЃџрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃАрЃљрЃќрЃарЃЊрЃћрЃџрЃљрЃЊ.

---

**рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃюрЃљрЃЉрЃўрЃ»рЃў:**

- [рЃерЃћрЃљрЃЊрЃљрЃарЃћ рЃърЃарЃЮрЃбрЃћрЃўрЃюрЃћрЃЉрЃў]
- [рЃЏрЃљрЃЕрЃЋрЃћрЃюрЃћ рЃЎрЃарЃћрЃљрЃбрЃўрЃюрЃћрЃЉрЃў]
- [рЃарЃЮрЃњрЃЮрЃа рЃЏрЃўрЃЋрЃўрЃдрЃЮ рЃърЃарЃЮрЃбрЃћрЃўрЃюрЃў рЃАрЃгрЃЮрЃарЃљрЃЊ?]
```

## рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃў

### рЃбрЃўрЃърЃЮрЃњрЃарЃљрЃцрЃўрЃљ
- **Bold** Рђћ рЃцрЃљрЃАрЃў, рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў, рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў
- *Italic* Рђћ рЃЉрЃарЃћрЃюрЃЊрЃў
- рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃў Рђћ рЃљрЃдрЃгрЃћрЃарЃљ
- `---` Рђћ рЃњрЃљрЃЏрЃДрЃЮрЃцрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА

### рЃАрЃўрЃњрЃарЃФрЃћ
- рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ: 1-2 рЃгрЃўрЃюрЃљрЃЊрЃљрЃЊрЃћрЃЉрЃљ
- рЃАрЃљрЃњрЃљрЃюрЃЏрЃљрЃюрЃљрЃЌрЃџрЃћрЃЉрЃџрЃЮ рЃЉрЃџрЃЮрЃЎрЃў: рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃБрЃЏ 3 рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃў
- рЃЏрЃЌрЃџрЃўрЃљрЃюрЃў рЃърЃљрЃАрЃБрЃ«рЃў: рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃБрЃЏ 400 рЃАрЃўрЃбрЃДрЃЋрЃљ
